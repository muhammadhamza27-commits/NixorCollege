# NixorCollege

A minimal HTML and CSS practice project for learning Git, GitHub, and basic web development structure.

## Overview

This repository contains a very small static web project with two simple pages:

- A root page (`index.html`) with a basic "Hello World" output
- A local practice page (`LocalRepo/index1.html`) with a paragraph styled by `LocalRepo/style.css`

The project is intentionally lightweight and focused on fundamentals.

## Why This Project Exists

NixorCollege appears to be a starter repository for practicing:

- Creating and organizing files in a web project
- Writing basic HTML and CSS
- Using Git for commits and GitHub for remote collaboration

## Project Structure

```text
NixorCollege/
|- index.html
|- README.md
`- LocalRepo/
	|- index1.html
	|- style.css
	`- README.md
```

## File Details

- `index.html`: Root demo page that currently renders `Hello World`
- `LocalRepo/index1.html`: Secondary demo page with sample text
- `LocalRepo/style.css`: Styles paragraph text to red
- `LocalRepo/README.md`: Basic note for the local folder

## Prerequisites

- A modern web browser (Edge, Chrome, Firefox, etc.)
- Git (optional, but recommended for version control practice)

## Getting Started

1. Clone the repository:

	```bash
	git clone https://github.com/muhammadhamza27-commits/NixorCollege.git
	cd NixorCollege
	```

2. Open the main page:

	- Open `index.html` in your browser

3. Open the local practice page:

	- Open `LocalRepo/index1.html` in your browser

## Usage

Use this repo as a sandbox for basic front-end and Git practice:

1. Edit text or structure in `index.html` or `LocalRepo/index1.html`
2. Update styles in `LocalRepo/style.css`
3. Commit and push your changes

## Current Output

- Root page text: `Hello World`
- Local page text: `This is a new repo`
- Paragraph styling: red text via CSS
